Needs Requested

This section allows you to filter requests displayed below.

Item Type

Use this to filter requests below by a specified item type. Options are foods, toiletries or other.

Food Type

Use this to filter requests below by food type. Options are Dairy, Grain, Meat, Fruit, and Vegetables.

Requests

This section displays all the client requests. Each request is displayed as a card, with the item type, food type, and description.

Edit Request

This button allows you to edit a request. It will take you to a request form, and automatically put in the request details. This button can be used to edit spelling and grammar errors in requests from clients. Once the requests are edited and submitted, they will be sent to the donor page.

Post Request

This button allows you to send a client request through to the donor page. Use this button when a request is ready to be sent to the donor page.

Delete Request

This button allows you to delete a client request. This button can be used in the case that a request is not meaningful or contains too many errors to be understood.
